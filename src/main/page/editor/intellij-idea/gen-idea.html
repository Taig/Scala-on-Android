gen-idea

It is now necessary to create the {{ names.abbreviation( 'ide' ) }} configuration files via the gen-idea {{ names.abbreviation( 'sbt' ) }} plugin. This step will hopefully soon be superfluous, but for now it is still obligatory.

Navigate your terminal to the project root directory and run {{ command( 'sbt gen-idea' ) }}. This will create {{ names.name( 'intellij-idea' ) }} 13 configuration files, but newer versions of {{ names.name( 'intellij-idea' ) }} are able to convert the configuration to their appropriate format.

You can now chose the Open option of the {{ names.name( 'intellij-idea' ) }} and select the root directory of your project.
