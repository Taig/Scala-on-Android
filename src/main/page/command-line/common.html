Common tasks

Below is a list of common and important {{ names.abbreviation( 'sbt' ) }}, and its respective {{ names.name( 'android' ) }} plugin, commands.

Command Description
android:run Compile the project, package a debug apk and deploy it to an active device or emulator
android:packageRelease Create a signed apk
android:packageDebug Create a debug apk
reload Apply changes that have been made to your {{ names.abbreviation( 'sbt' ) }} configuration
compile Compile the source code
clean Remove all compiled and generated code, delete {{ names.name( 'proguard' ) }} cache. This might be useful when experiencing issues with incremental builds.

If you find yourself running a particular {{ command( 'android:*' ) }} task very often, you may want to add a shortcut to your {{ names.abbreviation( 'sbt' ) }} configuration. E.g. {{ command( 'run <<= run in Android', 'scala' ) }} allows you to trigger the task from the console via {{ command( 'run' ) }} instead of {{ command( 'android:run' ) }}.
