Plugins

After startup make sure to first install the {{ names.name( 'scala' ) }} and {{ names.abbr( 'sbt' ) }} plugins as shown below.

{{ image( '/asset/image/android-studio-start-screen-configure.png', 'On the start screen click Configure' ) }} {{ image( '/asset/image/android-studio-start-screen-plugins.png', 'Click on Plugins' ) }} {{ image( '/asset/image/android-studio-start-screen-plugins-repositories.png', 'Click on Browse repositories' ) }} {{ image( '/asset/image/android-studio-start-screen-plugins-scala.png', 'Search for scala and install the Scala (custom language) plugin' ) }} {{ image( '/asset/image/android-studio-start-screen-plugins-sbt.png', 'Search for sbt and install the SBT (build) plugin' ) }}

Now close the plugins configuration and restart the {{ names.abbr( 'ide' ) }} to apply the changes.
