Library dependencies

The preferred way to add dependencies are managed dependencies. They offer fine-grained control about the dependencies, yet they are still easy to maintain. Library dependencies are maintained via the {{ names.abbreviation( 'sbt' ) }} configuration key {{ command( 'libraryDependencies' ) }}. The {{ names.name( 'play' ) }} documentation provides a good {{ sourcelink( 'play-framework-documentation-sbt-dependencies', 'introduction' ) }} to managed, unmanaged dependencies and resolves.
