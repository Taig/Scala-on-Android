Manifest

Some manifest values (e.g. the {{ command( 'version' ) }} and {{ command( 'targetSdkVersion' ) }}) can be specified in the {{ names.abbreviation( 'sbt' ) }} configuration file. The {{ names.name( 'sbt-plugin' ) }} will then inject the values into the existing AndroidManifest.xml and save it at {{ command( './target/android-bin/AndroidManifest.xml' ) }}. If you don't let {{ names.name( 'android-studio' ) }} know about the patched manifest file, the {{ names.abbreviation( 'ide' ) }} might annoy you with linting messages that some {{ names.abbreviation( 'api' ) }} calls are not available with a {{ command( 'minSdkVersion' ) }} of 0.
