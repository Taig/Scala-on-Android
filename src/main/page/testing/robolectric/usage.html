Usage

Place your unit tests in the {{ command( './src/test/scala' ) }} directory. All test classes may be equipped with an {{ command( '@Config' ) }} annotation for {{ names.name( 'robolectric' ) }}. The annotation allows you to manually specify the {{ command( 'AndroidManifest.xml' ) }} file location or {{ names.abbr( 'android-sdk' ) }} version to use for text execution.
